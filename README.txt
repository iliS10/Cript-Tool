--------------------------------CRYPTO TOOL USER GUIDE--------------------------------------

FRANCAIS:

///INFO
CryptoTool est un programme de cryptage en XOR fait avec python et converti en C++
Vous pouvez également décrypter votre fichier de la meme methode que celle du cryptage 
------------------------------------
Fait par https://github.com/iliS10
-------------------------------------
///Utilisation

1/Faire un fichier texte avec la phrase que vous voulez crypter

2/Pour le convertir en fichier vierge faites clique droit sur votre fichier texte puis Propriétés ensuite enlevez le .txt a la fin

3/Ouvrir Crypt Tool.exe

2/Entrer le nom du fichier mis dans le dossier de Crypt Tool

3/Entrer le nom du fichier qui sera crypté et accessible dans le dossier du logiciel

4/Entrer une clé pour votre dossier que vous aurez choisi

5/Et voila ! votre dossier a bien été cypter en XOR 

Vous pouvez décrypter votre fichier en faisant la même methode mais en utilisant votre fichier crypté et votre clé utilisé.

ENGLISH:

///INFO
CryptoTool is an XOR encryption program made with python and converted to C++
You can also decrypt your file using the same method as encryption
------------------------------------
Made by https://github.com/iliS10
-------------------------------------
///Use

1/Make a text file with the sentence you want to encrypt

2/2/To convert it to a blank file, right click on your text file then Properties then remove the .txt at the end

3/Open Crypt Tool.exe

2/Enter the name of the file put in the Crypt Tool folder

3/Enter the name of the file which will be encrypted and accessible in the software folder

4/Enter a key for your chosen folder

5/There you go! your file has been encrypted in XOR

You can decrypt your file by doing the same method but using your encrypted file and your used key.
